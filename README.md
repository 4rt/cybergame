# cybergame

### Docker Setup
```
docker-compose up -d --build
```

#### App runs
```
http://localhost:8081/
```


### Manual Setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### App runs
```
http://localhost:8080/
```

#### Run your end-to-end tests
```
npm run test:e2e
```

#### Run your unit tests
```
npm run test:unit
```
